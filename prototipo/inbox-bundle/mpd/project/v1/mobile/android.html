MPD · adaptação mobile

Seis telas para o celular Android do gabinete.

Mobile foca em uso em campo: captura no momento, consulta entre compromissos, devolutiva pelo bolso. Tabelas viram cards; sidebar vira tab bar inferior; data da timeline vira data acima do bloco. A captura rápida deixa de ser modal e vira tela própria (a mais usada). Mesma paleta: papel cru, tinta navy, bordô para urgência, serif para o relato.
