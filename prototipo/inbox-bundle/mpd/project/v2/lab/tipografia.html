Atenção: o UI inteiro agora está em Geist Mono

Observe o label do campo, o texto do input, o botão "Salvar demanda" e o tipo das interações na timeline. Não é mono só nos IDs — é mono em todo o chrome do produto. Se ficar incômodo (terminal-cosplay), volto para uma sans normal no UI e a mono só nos IDs. Se a ideia te seduzir, fica esse contraste forte: serif no relato (voz do cidadão) × mono no comando (voz do operador) × display no h1 (a pauta).

Quando decidir o display, diga "vai com Funnel" / "Instrument" / "Fraunces" / "Bricolage" — e se vai manter Geist Mono no UI ou voltar para sans neutra. Aí propago.
